# Customer Churn Data Science Project

This repository contains a data science project focused on
understanding customer churn behavior using a structured approach.

## Week 1 – Data Cleaning
- Missing value handling
- Data consistency checks
- Preparing a clean dataset

## Week 2 – Data Visualization & Chatbot
- Churn distribution analysis
- Feature-based visualizations
- Development of a basic chatbot for churn-related queries

## Week 3 – Exploratory Data Analysis (EDA)
- Dataset overview and understanding
- Univariate, bivariate, and trivariate analysis
- Correlation analysis
- Business insight generation

- ## Week 4 – Feature Engineering
- Derived new features related to customer behavior
- Encoded categorical variables
- Scaled important numerical features
- Prepared data for future modeling

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn
